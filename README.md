# AI-Powered-Carrier-Guidance-System-Enhanced
In this project we made a dashboard where student can easily choose his career guidance. In this app we integrate an A.I. Chatbot to interact and make easy to understand.
In this project we are three members one from data Science and the other 2 from A.I. background.
We use data analytics to clean data and feature engineering to make feature selection and make new features to make our model good accuracy. We used SVM, Linear regression, tf-idf and embedding and some more ML tools rather than this we use skills like full stack to build dashboard.
And done all analysis of all ML Models and showed complete trend using this Model.
